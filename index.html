Classificação de Flores com KNN

Dataset Íris

Problema

  • Classificar flores automaticamente
  • Baseado em medidas

O objetivo é prever a espécie da flor com base em características físicas.

Dataset Íris

  • Espécies:
    • Setosa
    • Versicolor
    • Virginica

  • Features:
    • Comprimento da sépala
    • Largura da sépala
    • Comprimento da pétala
    • Largura da pétala

Tipo de Problema

Classificação Supervisionada

Temos dados com resposta, então usamos aprendizado supervisionado.

Aplicando KNN

  • Escolher K
  • Calcular distâncias
  • Selecionar vizinhos
  • Classificar por maioria

Código (sklearn)


from sklearn.neighbors import KNeighborsClassifier

model = KNeighborsClassifier(n_neighbors=3)
model.fit(X_train, y_train)
y_pred = model.predict(X_test)
